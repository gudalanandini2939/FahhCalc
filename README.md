# 🧮 FahhCalc

![FahhCalc Home](screenshot.png)

A modern and interactive calculator built using **HTML, CSS, and JavaScript**. FahhCalc combines everyday calculations with engaging math challenges, multiple game modes, sound effects, themes, and a fully responsive user interface.

---

## ✨ Features

### 🧮 Calculator
- Basic arithmetic operations
- Cursor editing
- Expression validation
- Keyboard support
- Clear and Delete functions
- Responsive calculator interface

### 🎮 Challenge Modes
- Question Challenge
- Time Challenge
- Survival Mode
- Endless Mode
- Boss Challenge
- Daily Challenge

### 🎯 Difficulty Levels
- Easy
- Medium
- Hard

### ❤️ Gameplay
- Lives System
- Timer
- Streak Counter
- Countdown before starting
- Pause & Resume
- Quit Confirmation
- Result Screen

### 📊 Statistics
- High Score
- Total Calculations
- Accuracy
- History Panel
- FAHH Count

### 🎨 User Interface
- Modern Glassmorphism Design
- Smart Drawer Menu
- Multiple Themes
- Smooth Animations
- Fully Responsive Design

### 🔊 Sound Effects
- Click Sound
- Correct Answer Sound
- FAHH Error Sound
- Sound Toggle
